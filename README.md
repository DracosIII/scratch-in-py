# scratch-in-py
# 🧩Programmation Visuelle

Une application éducative de programmation visuelle inspirée de Scratch, conçue pour apprendre les bases de l'algorithmique et de la logique de manière interactive.

Ce projet a été développé en Python avec l'interface graphique **PyQt6**. Le but du jeu est de guider un cube bleu vers une cible rouge en empilant des blocs d'instructions (mouvements, boucles).

## 🚀 Fonctionnalités

- **Interface drag & drop (clic) :** Sélectionnez des blocs d'actions pour construire votre script.
- **Blocs de mouvement :** Avancer, Reculer, Monter, Descendre.
- **Blocs de logique :** Intégration de boucles (`While`) paramétrables pour répéter une séquence d'actions.
- **Animation en temps réel :** Exécution visuelle du code sur une grille quadrillée.
- **Système d'essais :** Le joueur dispose d'un nombre limité de tentatives pour atteindre l'objectif.

## 🛠️ Technologies utilisées

- **Langage :** Python 3.x
- **Interface Graphique :** PyQt6
- **Architecture :** Programmation Orientée Objet (POO)
